This project is a front-end test build for Covalent Logic. It recreates the login and dashboard access page for the State of Louisiana Physical Therapy Board based on the provided mockups.

## Features

- Responsive layout
- Mobile hamburger navigation
- User dropdown menu (single-level)
- Search overlay with accessible form
- Login form with standard browser validation
- Password visibility toggle

## Requirements Addressed

- **Semantic, ADA-conscious markup** using proper headings, labels, buttons, roles, and ARIA attributes.
- **Fully responsive build**, optimized from large desktop down to mobile.
- **Single-level primary navigation** with a user icon flyout that mirrors the provided mockup.
- **Search overlay** triggered by the magnifying glass button, with accessible focus handling.
- **Standard browser validation** used on form fields (e.g., `required`, input types).
- **Standard HTML form submission**, no custom API handling required.

## Technologies

- HTML
- CSS
- JavaScript
- Font Awesome
- Google Fonts (Roboto)

## Notes

This build focuses on clean structure, accessibility considerations, and accurately translating the provided design into a functional front-end prototype.
